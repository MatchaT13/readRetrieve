1. You will need to have JDK 8 or higher installed.
2. Use standard Java libraries.
3. Place the 4020a1-datasets.xml file in the same directory as the Java files.
4. You can adjust the querying speed by manipulating the thread. sleep code, but 100ms is the recommended time, as PubMed can only take 10 requests per second max. 
5. Run the code.
